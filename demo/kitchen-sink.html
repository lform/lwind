What is the Gutenberg editor?

Gutenberg is the block-based editor introduced in WordPress 5.0. Each piece of content is a discrete block that can be moved, configured, and styled independently.

Can I customize block styles?

Yes — all block styles in this file are baseline defaults meant to be overridden on a per-project basis downstream.

Code

const blocks = ['paragraph', 'heading', 'quote', 'image'];
blocks.forEach(block => register(block));
